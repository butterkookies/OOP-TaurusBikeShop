I. Project Overview 
Students are required to design and implement a database system based on an approved case 
study. The project must demonstrate proper database design, normalization, and correct SQL 
implementation within a working application. 
 
II. Minimum Technical Requirement 
The project must use a relational database management system (RDBMS), with Microsoft SQL 
Server (MSSQL Server) as the minimum requirement. The application must successfully connect and 
operate with the chosen DBMS, demonstrating proper database configuration and reliable connectivity 
throughout the system.
